# UAV Secure Video Pipeline 🚁🛡️

**UAV Secure Video Pipeline** 是一個專為無人機 (UAV) 監控與蒐證影像設計的高效防篡改安全框架。

在現今 Deepfake 與影像編輯技術氾濫的時代，本專案結合了**物件偵測(ROI)**、**密碼學數位簽章(Ed25519)** 與 **資訊隱寫術(LSB 數位浮水印)**，為錄影畫面建立「拍下即鎖定」的最高層級防護，確保「關鍵證據」不容任何變造。

---

## ✨ 核心特色與機制

- 🎯 **智慧 ROI 區分**: 系統透過 `CEASCLiteROI` 智慧區分畫面中的「重點保護目標(ROI)」與「背景」。
- 🔐 **不可見之防偽浮水印**: 將無人機設備 ID、時間戳記、畫格索引以及「ROI區域之 SHA-256 雜湊碼」，透過 Ed25519 私鑰簽署後，悄悄以 LSB (Least Significant Bit) 形式藏入影像「背景」的最低位元中。
- ⚖️ **正規化技術**: 獨創的 3-bit 影像正規化 (`canonical_utils.py`)，保證了浮水印嵌入前後主體結構完全一致，克服了傳統浮水印容易自毀 Hash 的技術瓶頸。
- 📊 **專為學術研究優化**: 執行加密錄影時，系統會同步輸出 `.csv` 檔案，自動紀錄每格影像的嵌入容量(Capacity/Payload)、ROI 佔比，以及不可見度指標 **PSNR** 與 **SSIM**，方便學術論文數據分析。
- 🕵️ **精準的第三方驗證**: 支援驗證工具讀入加密影片，藉由解開背景 LSB 取出公鑰與數位簽章比對，能精準「抓包」影片在第幾格、甚至是極微小的局部竄改。

---

## 🛠️ 專案檔案結構介紹

* `main.py` - 主程式入口，啟動相機自動加上防偽簽章並錄製為 FFV1 無損壓縮影片。
* `verify_video.py` - 第三方驗證管線，負責拆解隱藏資料、檢查憑證、並比對影像雜湊值以發現變造。
* `lsb_embed.py` - 隱函數庫，負責把二進位資料藏進背景像素的後 3-bits。
* `canonical_utils.py` - 正規化演算法工具。
* `signature_utils.py` - Ed25519 簽章與金鑰產生管理庫。
* `tamper_video.py` / `gui_tamper.py` - 專為測試防偽能力的**竄改模擬工具/圖形介面**。

## 🚀 快速上手使用說明

### 1. 安裝環境依賴需求
請確保您的環境安裝了 Python 3 與相關 AI 及視覺套件：
```bash
pip install opencv-python numpy Pillow
# 如果系統本身還有其他依賴套件 (如 torch 用於推論) 請一併安裝
```

### 2. 啟動無人機加密安全管線
開啟終端機並執行以下指令，系統會擷取您的攝影機畫面，進行即時影像簽署與浮水印隱形疊加。
```bash
python main.py
```
> 💡 **Tip:** 執行完畢後，除了會獲得 `uav_signed_xxxx.avi` 影片外，還會同步產生一份 `uav_signed_xxxx.csv`，內含所有論文撰寫會用到的 PSNR、嵌入位元等重要指標資料。

### 3. 模擬「人為竄改」(測試用)
為了證明這套系統的安全有效性，專案內準備了一個**局部竄改模擬器**。這支工具刻意只改變畫面中小部分的座標，完美保留背景其餘 LSB 浮水印。

您可以直接開啟直覺化的 UI 介面：
```bash
python gui_tamper.py
```
在介面中：
- 選取您剛剛產生受保護的 `.avi` 影片。
- 設定你要縮放的**假飛機/物體圖片**大小。
- 指定造假物體要在**第幾個畫格區間**出現，點擊「開始竄改處理」即可生成 `tampered.avi` 影片檔案。

### 4. 進行第三方影像鑑識與驗證
我們提供了一套完整的圖形驗證介面工具，讓鑑識人員可以輕鬆查驗影片是否遭受變造：
```bash
python verify_gui.py
```
打開介面後，您可以：
- **選擇驗證模式**：可驗證「單支影片」，或是指定「整個資料夾」以進行批次連續驗證。
- **匯入公鑰檔 (.pem)**：載入系統對應的公鑰 (預設通常位於 `keys/` 資料夾下)，用來確認數位簽章是否為授權設備所發出。
- **時間範圍篩選**：若影片過長，您可以指定「只驗證特定秒數範圍」，加速作業流程。
- **產出鑑識報告**：點擊「開始驗證」後，畫面會即時滾動顯示逐格排查進度與「通過/失敗」的畫格數。系統最後更會自動把詳細結果打包為 `.json` 格式，匯出到 `verify_reports` 資料夾，作為具公信力的鑑識分析報告！

## 📑 應用情境與論文方向

本研究模型適用於「警用執法密錄器」、「證據保全空拍系統」、「即時遠距視訊驗證」。在撰寫相關文章時，您可以強調：
- **Capacity / Robustness**：由於我們利用了背景的大空間而保護了 ROI，擁有比起傳統全幅 LSB 更大的 Payload 空間。
- **Imperceptibility**：藉由 CSV 自動輸出的 PSNR 數據證明影像畫質肉眼無法分辨。
- **Security**：非對稱加密(Ed25519)保證了資料來源合法性與不可否認性。
